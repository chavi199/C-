# Product Overview

## Project Purpose
MishloachManot is a C++ application for managing Purim gift packages (Mishloach Manot). The system handles the creation, tracking, and management of gift packages sent between people, including product details, addresses, and kosher certification tracking.

## Key Features
- **Gift Package Management**: Create and manage Mishloach Manot packages with sender/recipient information
- **Product Tracking**: Track individual products with pricing, quantities, and kosher certification (Badats, Rubin, Lando)
- **Address Management**: Store and manage delivery addresses with city, street, and house number
- **Price Calculation**: Automatic calculation of total package costs including design and tool prices
- **Kashrut Verification**: Ensure all products in a package meet specific kosher certification standards
- **Gift Extensions**: Support for special gift items with size classifications (Small, Medium, Large)
- **Comparison Operations**: Compare packages by price and equality

## Target Users
- Individuals organizing Purim gift distributions
- Community organizations managing large-scale Mishloach Manot programs
- Users requiring kosher certification tracking for food gifts
- Anyone needing structured gift package management with cost tracking

## Use Cases
- Creating personalized gift packages with multiple products
- Tracking total costs for budgeting purposes
- Ensuring consistent kashrut standards across all products in a package
- Managing delivery addresses for multiple recipients
- Comparing package values between different recipients
- Organizing community-wide Purim gift distributions
